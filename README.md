# SQL Portfolio

This repository contains a structured collection of SQL scripts and exercises, organized by topic and difficulty.

## Structure

- **01_Basic_Queries/**: Fundamental SQL concepts including filtering, sorting, aggregations, and basic functions.
    - `Filtering_and_Sorting/`: WHERE, LIKE, ORDER BY, LIMIT
    - `Aggregations/`: SUM, AVG, COUNT, GROUP BY, HAVING
    - `String_and_Date_Functions/`: CONCAT, SUBSTRING, DATEDIFF, DATE_TRUNC

- **02_Joins_and_Sets/**: Intermediate techniques for combining data from multiple sources.
    - `Join_Types/`: Inner, Left, Full Outer, Self Joins
    - `Set_Operations/`: UNION, UNION ALL, INTERSECT
    - `Case_Logic/`: Complex CASE WHEN transformations

- **03_Advanced_Techniques/**: Advanced SQL features for complex data analysis.
    - `CTEs/`: Common Table Expressions for readability
    - `Subqueries/`: Scalar and correlated subqueries
    - `Window_Functions/`: ROW_NUMBER, RANK, LEAD/LAG

- **04_Database_Design_and_DDL/**: Database management, schema design, and optimization.
    - `Schema_Creation/`: CREATE TABLE, Constraints (PK/FK)
    - `Data_Cleaning_Scripts/`: UPDATE, DELETE, TRUNCATE, COALESCE
    - `Optimization/`: INDEX creation and EXPLAIN plans

## Usage

Each directory contains scripts relevant to the specific topic. Refer to the comments within each script for details on the logic and usage.
